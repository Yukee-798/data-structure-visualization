## 项目结构
```
|-- public
|-- src
    |-- api 规范请求格式
    |-- components 封装页面通用组件
    |-- pages 组件的容器，同时对应路由下的页面
    |-- router 路由配置
    |-- store
    |-- types 存放全局复用的 type
    |-- utils 封装通用函数、hooks
```

## 技术栈

* TypeScript
* node.js
* react
* react-spring 动画库
* react-three-fiber 构建 3d 模型
* mbox 用于数据管理
* antd UI库
